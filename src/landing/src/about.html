About Emissary

Emissary is a free, easy-to-use local messenger. It is intended for use on public devices such as library or school computers, but it can also be used at home on a shared computer or similar device. See information on getting started below

Getting started

Getting started with Emissary is super simple:

  1. First, create an account and make sure to write down your word key.
  2. Next, generate a new route with the + icon in the top right of the routes list.
  3. Give the route word key to someone you want to chat with.
  4. When you send the first message via the route, your profile picture and username will show up for the other user (and vice versa).
  5. Happy chatting!
